Question-2: What is the difference between innerHTML, innerText, and textContent?
Ans:innerHTML returns the HTML content inside an element, innerText returns the visible text as displayed in the browser, excluding hidden elements and textContent returns the text content of an element, including hidden text, but treats everything as plain text.


Question-3: What is event delegation in the DOM?
Ans: event delegation in the Dom is where single event listener is added to a parent element to handle events for multiple child elements.


Question-4: What is event bubbling in the DOM? Ans: event bubbling in the DOM works from the target element and finish at the root element.


Question-5: How do you create, add, and remove elements using JavaScript? Ans: create a new element using the document.createElement() method. Once an element is created, we can insert it into the DOM using different methods like appendChild(). and we can remove an element using removeChild() or remove().
